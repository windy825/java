# 연산자

```
연산자 : 연산을 수행하는 기호
피연산자 : 연산자의 연산 수행 대상
"모든 연산자는 연산결과를 반환한다."
```

- **연산자의 종류**

  |    종류     |             연산자              |              설명              |
  | :---------: | :-----------------------------: | :----------------------------: |
  | 산술 연산자 |     + - * /  %       <<  >>     |     사칙연산, 나머지, 비트     |
  | 비교 연산자 |      >  <  >=  <=  ==  !=       |      대소 비교, 같고 다름      |
  | 논리 연산자 | &&  \|\|  !         &  \|  ^  ~ |       and,or,not,  비트        |
  | 대입 연산자 |                =                |    우변의 값을 좌변에 저장     |
  |    기타     |     (type)   ?:  instanceof     | 형변환, 삼항(조건), 객체지향쪽 |
